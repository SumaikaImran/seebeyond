#  See Beyond – Guided by Sound, Empowered by Touch

An AI-powered assistive system designed to help visually impaired individuals navigate their surroundings and perform daily tasks independently using Computer Vision, Artificial Intelligence, and Voice Interaction.

##  Features

- Real-time Object & Obstacle Detection
- Face Recognition
- OCR (Text Recognition)
- Pakistani Currency Detection
- Voice Commands (Speech-to-Text)
- Audio Feedback (Text-to-Speech)
- Flutter Mobile Application
- FastAPI Backend
-  Real-time Camera Processing


## Tech Stack

- **Languages:** Python, Dart
- **Computer Vision:** OpenCV, YOLOv8, FaceNet, PaddleOCR
- **Frameworks:** FastAPI, Flutter
- **AI/ML:** PyTorch, TensorFlow Lite
- **Database:** PostgreSQL / SQLite
